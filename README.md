# learningNotes说明

## 0 仓库说明

1. 仓库主要用于存放平时的学习笔记，笔记中如果有侵权等问题，请及时联系本人;
2. 每个一级目录下包含一项学习内容，可能包含书籍，引用图片，源文件等内容；
3. 学习笔记采用markdown进行编写,下载后建议通过Typora工具进行查看；
